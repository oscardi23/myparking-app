live_help Preguntas Frecuentes

¿Que es MyParking? expand_more

Es una herramienta web para gestionar la operacion diaria de un parqueadero.Controlar entradas y salidas, ver la ocupación y llevar el control de ingresos desde un solo lugar.

¿Necesito internet para usar la app? expand_more

No.Funciona directamente desde el navegador web, no requiere instalación.

¿Puedo usarlo desde el celular? expand_more

Si, se puede usar desde celulares, tablets o computadores con acceso a internet.

¿Funciona para carros y motos? expand_more

Si, puede registrar diferentes tipos de vehículos.

¿La informacion esta segura? expand_more

Si, los datos se almacenan de forma segura y solo son accesibles desde tu cuenta.

¿Puedo gestionar más de un parqueadero? expand_more

Por ahora, cada cuenta está pensada para un solo parqueadero. Más adelante se habilitará la gestión de múltiples ubicaciones.

¿Necesito conocimientos técnicos para usarlo? expand_more

No. La plataforma está diseñada para ser intuitiva y fácil de usar, sin necesidad de conocimientos técnicos previos.
